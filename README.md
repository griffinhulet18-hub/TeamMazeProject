# Team Maze Project

# Project Description
Team Maze is a Java-based project that generates and solves mazes. Users can explore maze-solving algorithms and see how the program works in an interactive way. This project helps us practice Java programming and teamwork.

## How to Run
### GUI Version
Run `Main.java`. Use arrow keys or WASD to move.  

### Console Version
Comment out the GUI line in `Main.java` and uncomment `MazeGame` code.
